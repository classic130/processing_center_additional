Not Logged into Windows 95
You are currently not logged into Windows 95. Please log on before proceeding.

If you have forgotten your password follow these steps:

Note: By following the steps below you will make all Digital IDs you currently have unusable. It will not affect other users on your machine. If you do not want to use this process contact your administrator.

  • Go to your Start Menu
  • Choose Programs / Window Explorer
  • Click on the Windows directory
  • Locate a file with .pwl extension. If there are multiple users on your machine you will have more than one .pwl file. Delete the one with your login name
  • Log onto Windows 95 and choose a new password
