Digital ID Could Not Be Installed

The most likely reasons for this problem are:
  • A Certificate chain could not be built to a trusted root authority.You do not have the Root Certificate installed in your system.The Root Certificate has to be installed in the Certificate Store "Trusted Root Certificate Authority".
  • The URL from which you are trying to enroll is not a Trusted Site. Make the URL a trusted site by adding the URL name to the list of trusted sites in the security Tab of the "Internet Options" dialog in the "Tools"
  • The browser you are using now is not the same one you used when you enrolled for your Digital ID. You must use the same browser
