Digital ID Could Not Be Generated

The most likely reasons for this problem are:
  • You did not choose to run the Enrollment ActiveX control When loading the enrollment page, you should have seen a "Security Warning" dialog for a program digitally signed by Microsoft. (If you did not see this dialog see the next point below). This program is required to enroll for a certificate and is safe to run on your computer and it's authenticity is proven by the digital signature on the program which was verified when you saw the dialog.
  • You may have turned Active Content off. The enrollment page uses an ActiveX control to do the certificate request generation. Turn on ActiveX by going to the "Security Settings" tab in the "Internet Options" dialog of the "View" menu in Internet Explorer. Set the security settings to "Medium", which is the Microsoft default security setting.
  • You may not be running the latest version of Internet Explorer. Contact your Managed PKI administrator to know the supported versions of Microsoft Internet Explorer.
