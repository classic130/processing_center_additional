Key Pair Generation Problem

Your Web browser did not properly generate a public key - private key pair.

These keys are needed in order to create a Digital ID. The most common reason for this problem is that after the "Submit" button was pressed, Navigator displayed a Password Entry window. If "Cancel" was clicked in that window, Navigator will not generate a key pair at that time.

To get a Digital ID, click the browser's "Back" button to return to the Enrollment form and press the "Submit" button again. Then, when the Password Entry window is displayed, enter a password and click "OK".
