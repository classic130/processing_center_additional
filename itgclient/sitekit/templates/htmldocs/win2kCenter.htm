Windows Resource Center

 
This page is intended for use by enterprise administrators. It gives information about how to enable end users on Windows 7/8.1 desktops who do not have power user or administrator privilages for using Go Secure! for Web Applications.

Administators can download the Microsoft Installer (MSI) packages for the Personal Trust Agent (PTA) for different languages under Deploying the PTA MSI packages.

Administators can download the Microsoft Installer (MSI) packages for the Go Secure! for Exchange for different languages under Deploying the Go Secure! for Exchange MSI packages.

Instructions for deploying the Symantec MSI packages can be obtained by clicking the appropriate links below.

  • Overview of Deploying MSI packages
  • Deploying the PTA MSI packages
  • Deploying the Go Secure! for Exchange MSI packages
