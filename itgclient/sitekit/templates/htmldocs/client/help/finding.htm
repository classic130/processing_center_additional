Finding a person's Digital ID

This section describes how to find someone's Digital ID, so that you can send them encrypted e- mail. The simplest way to get someone's Digital ID is for them to send you a signed e-mail message. When they do this, their e-mail application attaches the Digital ID to the message. Your Web browser will automatically download the Digital ID into its list.

However, if you know someone has a Digital ID, you can use the Digital ID Center to find their ID and install it in your Web browser.

To find a Digital ID:

  1. Go to the Digital ID Center and select the Search form by clicking the Search button at the top of the page.
  2. Enter the e-mail address or name for the person whose ID you want. Click the Search button.
  3. You either see a list of Digital IDs that match or you will get a message saying no IDs were found. Click the link for the Digital ID that you want to download. If you did not get a list of IDs, return to the Search page and try another e-mail address or name. The Digital ID Center only finds exact matches, so you might try a person's nickname.
  4. Once you click the ID, the page displays more information about the Digital ID. Make sure this is the correct ID you want, and then click the Download button.
  5. Choose the download option for your Web browser that begins with the text "Someone else's Digital ID," and then click the button called Download This Certificate.
  6. The Digital ID is downloaded into your Web browser, and then you see a message confirming that the ID was downloaded.
You can now send encrypted e-mail to the person whose ID you downloaded. For more information on sending encrypted e-mail and signing messages, see the How To section of this online help system.
