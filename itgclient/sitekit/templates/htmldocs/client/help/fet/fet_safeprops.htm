Safe Properties

The Safe itself has properties that you can set. These properties affect the way the Safe works. For example, they determine how often the Safe asks you to type in your password. They also specify where encrypted files are stored: in the Safe or in their original location.

In addition to Safe properties, each file in the Safe has its own file properties.

To set the Safe's properties:

  1. Open the Safe and enter your password.
  2. Make sure no files are selected in the Safe.

  3. Choose File > Properties from the Safe menu. The following dialog box appears.

  4. Use the tabs and options to make changes to the Safe properties. The tabs are described below.
You can also view the Safe properties by right-click the Safe icon on the desktop, or the Safe icon which appears in the Windows Explorer left pane or by right click on the view which shows the encrypted files.

General properties let you change the Digital IDs that the Safe uses for encryption and signing. The tab also displays the amount of disk space used by the files in the Safe.

Advanced properties set where the Safe stores encrypted files. From this tab, you can also change the Safe password and set advanced security properties.

Filters let you exclude certain file types or folders from encryption.

Digital ID properties let you manage your Digital ID. If you do not have a Digital ID, you can request one from this tab.
