Advanced Properties

The Advanced Properties determine how files are moved into the Safe and how often you are prompted for the Safe password.

To change Advanced properties:

  1. Open the Safe properties.
  2. Click the Advanced tab.
  3. Change any settings you want and then click OK. The settings are described below.

Location of Encrypted File :

  • Move file to Safe - Any encrypted file is placed in the Safe. The Safe securely deletes the original file. When you decrypt the file, it appears back in its original folder.
  • Encrypt inplace - Any encrypted file remains in the original location of the file but it gets a new .ENC extension. For example, if you encrypt a file named file.txt, the new file name is file.txt.enc. Using Explorer, you can right-click the file to decrypt it and then double-click the file to run it. In-place decryption also securely deletes the original file.

    Password Options :

  • Prompt everytime - This configures the Safe to ask you for the password every time you encrypt or decrypt a file or change any properties.
  • Prompt after XX minutes - This sets a number of minutes during which you do not have to enter the password. For example, if you set the password timeout to 5 minutes, you can encrypt and decrypt files, but if you stop working with the Safe for five minutes or more, the Safe asks you to re-enter the password the next time you try to do anything in the Safe. If you want maximum security, set the password to always prompt you. If you choose a timeout instead, make the number of minutes small.

    In addition to the password protection provided by the Safe you can also set up a password protected Windows screen saver.

    Change Password
    This button lets you change the password that you use to unlock the Safe.

    Security
    This button lets you set security options and modify the Windows screen saver.


    Note : If you are using a certificate installed in the Symantec CSP then the password will be prompted everytime you access your private key or everytime the time-out occurs, depending on your settings. Also note that the Change Password option is not available in this case.
