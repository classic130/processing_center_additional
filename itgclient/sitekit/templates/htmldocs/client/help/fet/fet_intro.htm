What is the Symantec File Encryption Tool?

The Symantec File Encryption Tool is a desktop utility that allows users to protect sensitive files on their machine using encryption. Files/Folders can be encrypted by dragging and dropping them on the Safe icon which appears on the desktop and also in the Windows Explorer left pane.

The Safe works by encrypting your files using a Digital ID. That is, the files are scrambled in such a way that no one could read them directly from your hard disk. When you want to work with a file, you remove it from the Safe using a password that only you know. Once you are done making changes, printing, or simply viewing the file, you return it to the Safe.
