General Properties

General Properties let you change the Digital ID that the Safe uses when encrypting files or folders. To get to these options, open the Safe properties.

Details -
This section lists the number of files in the Safe and the amount of hard-disk space they are using.

Digital ID for Encryption and Signatures -
This section lists the person who owns the Digital ID and the expiry date for that ID. It also shows the serial number to uniquely identify the ID. You can select a different Digital ID by clicking the Change button, which starts the Personal Trust Agent (PTA). Using the PTA, you can choose a different Digital ID. Any files encrypted now on will use the new ID, any files which were already in the Safe before will be encrypted using the old ID. However if such a file is decrypted and re-encrypted it will use the new ID. If the Signature option is enabled then the files will also be digitally signed. This option will be enabled if the selected Digital ID can be used for signatures and the administrator has enabled this option for the user.
