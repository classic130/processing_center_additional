Digital ID Properties

To get to the Digital ID properties, go to the Safe properties and click the Digital ID tab. The Safe works by encrypting your files using a Digital ID. If you do not have a Digital ID or need to manage an existing Digital ID, use the buttons on the Digital ID Properties page to go to the Certificate Authority for your company. You must have a connection to your local network or web access to use these buttons.

Get Digital ID -
This button takes you to a web page where you can enroll for a Digital ID. This process involves filling out forms, receiving an email message, and then installing the Digital ID in your web browser. To use the Digital ID in the Safe, you need to install the ID in Internet Explorer. If you use Firefox, you can export the Digital ID, and then install it in Internet Explorer.

Revoke Digital ID -
This button lets you revoke a Digital ID. You might need to revoke a Digital ID if it was compromised in any way. For example, if someone learns your private key password, they could use your Digital ID to view any files you encrypted with it.

Renew a Digital ID -
This option is not implemented.

Recover a Digital ID -
This option is not implemented.
