Changing the Safe Password



The Safe uses a password that lets you unlock the Safe and access the files in the Safe. You can think of the Safe password as you would think of a key or a combination on a physical safe.

To change the Safe Password:
  1. Open the Safe and enter your password.
  2. Open the Safe properties.
  3. Click the Advanced tab.
  4. Click the Change Password button. The following dialog box appears.

  5. Type your old password in the top field. Type the new password twice.
  6. Click Browse to select a path and filename for the recovery file. This file helps you recover the password if you forget it. See Recovering the Safe Password for more information.
  7. Click OK to use your new Safe password.
For information on password options, such as how often the Safe will prompt you for the password, see Advanced Settings.

Note : The change password option is available only if you are using a certificate installed in the Microsoft CSP.
