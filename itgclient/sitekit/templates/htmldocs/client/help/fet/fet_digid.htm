Setting up the Safe's Digital ID

This page tells you how to set up the Digital ID that the Safe uses when encrypting or signing files. Before you can begin encrypting files, you must first set up a Digital ID. The Safe assumes you already have a Digital ID installed in Internet Explorer. If you do not have a Digital ID, you need to enroll for one.

To set up the Digital ID:

  1. Open the Safe and enter your password.
  2. Open the Safe properties.
  3. Click the Change button.
  4. Choose the Digital ID you want to use, and then click OK. If you have only one ID, it will be selected for you.
  5. When you change the Digital ID you may be switching from a certificate which is installed in the Microsoft CSP to one which is installed in the Symantec CSP or vice versa. In this case you may be prompted to enter the profile password or a new password for working with the Safe.


Tip : When you change the encryption ID, from that point on all encryption is done using the new ID. Any files encrypted with the old ID will remain encrypted with that ID until you decrypt and re-encrypt the files. For this reason, you should not delete an expired ID until you are sure that no files are encrypted using that ID.
