Recovering the Safe Password

If you forget the password to the Safe, you can use the password recovery feature to open the Safe again. After you open the Safe, you have to immediately set up a new password for the Safe. The password recovery feature helps you to open the Safe and setup a new password, it cannot recover the old password.

A Password Recovery Request file is created automatically when you change the password and specify a password during install You also have the option to save this recovery file at any other location you want such as a floppy disk. You will use this file later during the password recovery process.

The Safe shows the Recover Password option in the File menu if you have entered the wrong password three times. When you click on this option the following dialog box appears -

  1. Create a password recovery request file. If you have already saved this to some location then skip to Step 2.
  2. Send the recovery file to the administrator. If you are not sure how to do this contact your administrator. The administrator will verify that your request is valid and create a Password Recovery Token file for you.
  3. Import the recovery token sent by the administrator.
  4. If the recovery token is successfully imported you will be prompted to enter a new password. If you do not enter a new password now you will not be able to access the Safe.
Note - The old password is not available to either the end user or the Administrator. Also the new password is set by the end user and not by the Administrator. Password recovery is available only when the user is using a certificate installed in the Microsoft CSP.
