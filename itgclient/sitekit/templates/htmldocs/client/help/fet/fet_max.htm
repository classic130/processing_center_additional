Maximizing Security Settings

To maximize the security on your computer, always exit the Safe application when you are not using it. You can also follow these suggestions:

  • Choose a good password for the Safe and change it at regular intervals. Choose a password that is difficult for others to guess. For example, do not use your birthdate or name of your pet. Use a mix of letters and numbers. You might choose a password such as Mbh12t! to mean "My baby has twelve teeth!" This is easy for you to remember but difficult for others to guess. See Changing the Safe Password.
  • Set up a Windows screen saver. You can do this using the Windows Control Panel or the Security properties. This prevents people from using your computer when you leave it.
  • Set up the Safe to always securely delete files. See Security settings for directions.
