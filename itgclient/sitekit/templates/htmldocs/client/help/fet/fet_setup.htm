Setting up the Safe

When you first install and run the Safe, you must choose a Digital ID for the Safe to use. If you need a Digital ID, you can enroll for one and install it in Internet Explorer, where the Safe gets its list of IDs to use.

After choosing a Digital ID, you are asked to enter a password for the Safe. This is the password used to open(unlock) the Safe, much like you need a key or combination for a physical safe. You can change the password anytime. If you forget the password, you can recover it with help from your Administrator. This section describes the various ways you can set up the Safe. Use these links to make sure your Safe is properly configured.

  • Setting up the Digital ID describes how to choose which Digitial ID the Safe uses when encrypting files. You must do this before you can begin using the Safe.
  • Changing the Safe password tells you how to change the password for the Safe. When you first installed the Safe and ran it, you had to enter a password. This Safe password is different from any password you set up for your Digital ID.
  • Setting Safe properties tells you how to set the options that control how the Safe works.
  • Maximizing Security describes the various settings in the Safe and in Internet Explorer that control the security of your files.
