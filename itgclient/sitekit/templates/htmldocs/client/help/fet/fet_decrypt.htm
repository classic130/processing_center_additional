Decrypting Files

There are several ways to decrypt a file or set of files. When you decrypt files, you might be prompted to enter your private key password as well as the Safe password, depending on your Internet Explorer security settings. When the files are decrypted, they are returned to their original folders.

If the files were encrypted with an ID that is currently expired, the Safe will decrypt the files using the old ID provided that it is still installed in Internet Explorer. If you removed the old ID, you need to reinstall it in Internet Explorer. If you do not have the old ID, you cannot decrypt the files.

Decrypting from the Safe
This method decrypts one or more files at a time.

  1. Open the Safe and enter your password.
  2. Select the files you want to decrypt. Use Ctrl or Shift + click to select multiple files.
  3. Choose File > Decrypt from the menu or right-click the files and choose Decrypt from the menu.
If the file is signed in addition to encrypted, you can choose to decrypt and verify the file. The verification checks the digital signature stored with the file.

You can also choose Open from either menu to decrypt the file(s) and then run them using the application associated with the file extension. For example, if you double-click a file called file.txt, the Safe decrypts the file and then opens it in your default text editor (such as Notepad). This method only works for files in the Safe; it does not work for files that were encrypted in place.

Decrypting using drag and drop
In the Safe, you can drag and drop files from the Encrypted section to the Recently Decrypted view, provided that view is displayed. When decrypted, the files appear in their original folder.

You can also drag and drop files from the Safe to a folder in Windows Explorer. The files are then decrypted in the folder where you drag them rather than in their original folders. The Safe's decrypted view shows the directory where the files were decrypted.

Using Windows Explorer
You can decrypt files from Explorer. This method assumes the files were encrypted in place and have the .ENC extension.

  1. Start Windows Explorer.
  2. Select any files you want to decrypt. Use Ctrl or Shift + click to select multiple files. The files should all have the .ENC file extension.
  3. Right-click the files and choose Decrypt from the pop-up menu.
