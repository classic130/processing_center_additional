Encrypting Files

There are several ways to encrypt a file or set of files. When you encrypt files, you might be prompted to enter the Safe password. When the files are encrypted, they are either stored in the Safe or they are stored in their original folders but with the .ENC extension. You can change this with the Advanced properties.

Using the Safe menu -
This method encrypts one or more files at a time.

  1. Open the Safe and enter your password.
  2. Make sure no files are selected in the Safe.

  3. In the Safe, choose File > Encrypt File from the menu.
  4. In the dialog box that appears, choose the file you want to encrypt. You can use Ctrl or Shift + click to select multiple files. Click Encrypt. The files will appear in the Encrypted section of the Safe.
Using Windows Explorer -
You can encrypt files two ways using Explorer.
  1. Start Windows Explorer.
  2. Select any files or folders you want to encrypt. Use Ctrl or Shift + click to select multiple files.
    • Drag and drop any files from the right pane into the Safe in the left pane.

    • You can also right-click a file. In the pop-up menu, choose Encrypt.
Encrypting from the Windows Desktop -
You can drag and drop files from either the Windows Desktop or from Windows Explorer to the Safe icon on the Desktop.
  1. Select the files you want to encrypt.
  2. Drag and drop them to the Safe. If the Safe is not open, you will be prompted for the Safe password.
Encrypting previously decrypted files -
This method assumes you have previously decrypted files. There are two ways to encrypt one or more previously decrypted files:
  • Select the files you want to encrypt and then drag them from the Decrypted section to the Encrypted section.


    Use Ctrl + click or Shift + click to select multiple files or folders.

  • Select the files you want to encrypt. Right-click the file(s) and choose Encrypt from the pop-up menu.
