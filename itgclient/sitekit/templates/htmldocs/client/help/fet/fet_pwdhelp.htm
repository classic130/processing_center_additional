The Safe Password

The Safe uses a password that lets you unlock the Safe and access the files in the Safe. You can think of the Safe password as you would think of a key or a combination on a physical safe. When you first run the Safe, you must create a password for it. If you forget the Safe password, you can recover it with the help of the person who administers your system.

New Password
Enter a password for the Safe and confirm it by typing it twice. Choose a good password for the Safe and change it at regular intervals. Choose a password that is difficult for others to guess. For example, do not use your birthdate or name of your pet. Use a mix of letters and numbers. You might choose a password such as Mbh12t! to mean "My baby has twelve teeth!" This is easy for you to remember but difficult for others to guess.

Password Recovery
The Safe creates a recovery file that lets you recover your password if you forget it. Without a recovery file, there is no way to open the Safe if you forget your password. You can select the folder where the recovery file is stored by clicking the Browse button.

The recovery file stores a hash of your Safe password in encrypted form using the public key from your Password Recovery Agent (a System Administrator in your organization). There is no way for anyone to use the recovery file to view your Safe password. Even the Password Recovery Agent cannot view the password. See Recovering the Safe Password for more information.


Note : When using a certificate installed in the Symantec CSP the password features of the Safe such as 'change password' and 'recover password' are not available to the user. In this case the user unlocks the Safe using the Profile password.
