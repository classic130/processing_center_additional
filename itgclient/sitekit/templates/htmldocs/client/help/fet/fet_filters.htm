Filters

Filters let you exclude files from encryption based on file types or attributes or the folders in which they belong.

To get to these properties, open the Safe properties and then click the Filters tab. If you have the Safe password set to always ask you, you will be prompted for the Safe password before you see the Properties dialog box.

Use the drop-down list to select the types of files you do not want to encrypt and then click the Add button. You will see a dialog box asking for more information about the exclusion. If you choose to filter by extension, you must then enter a file extension (such as .EXE). If you filter by file attributes, check the attributes you want (read-only, hidden, and so on). If you filter by folders, choose the folders you do not want to encrypt.

Each type you choose is a separate exclusion. For example, if you choose a file extension (such as .EXE) and attributes (such as read-only), then files that are fit either type are not encrypted. That is, a file does not have to fit every type you choose; it need only match one of the criteria.

To remove a filter, select it and click Remove.
