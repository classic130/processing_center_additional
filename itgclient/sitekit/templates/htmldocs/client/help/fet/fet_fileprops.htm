File Properties

You can set properties for individual files in the Safe. There are several ways to view the properties for a file. First, select one or more files in the Encrypted section of the Safe. Use Ctrl + click or Shift + click to select the files you want. Then do one of the following:

  • Right click the file(s) and choose Properties from the pop-up menu, as shown below.
  • Click the Properties button in the toolbar.
  • Choose File > Properties from the menu.
The dialog box that appears shows the properties for the file(s).

The File Properties dialog box lists the file type, its original size and encrypted size on the hard disk, and when the file was created and encrypted. You cannot change these properties. The attributes for the file(s) show that the file is encrypted and/or signed. These attributes cannot be changed. Signing however can be enabled/disabled for all the files through the General Properties.


The following attributes allow you to easily work with files without having to manually move them in and out of the Safe each time you want to work with them. These attributes can be modified.

  • Encrypt when Safe is locked encrypts the file(s) when you lock the Safe.
  • Decrypt when Safe is unlocked decrypts the file(s) when you unlock the Safe.

    • You can use both of these options together or you can use one at a time. For example, if you are working on a set of files for a project, you might choose to encrypt all of the project files when you lock the Safe and then have them decrypt when you unlock the Safe. This provides an easy way for you to work with your encrypted files.
