A Brief Tour of the Safe

This page presents an overview of the Safe user interface and introduces some of the features of the Safe. Before you can use the Safe, you need to set up a Digital ID. See Setting up the Digital ID for directions.

The Safe interface resembles the Windows Explorer interface. You can view encrypted and decrypted files using the same organization as Explorer. For example, you can choose to view files with large or small icons. You can view all of the file details, such as size and time the file was encrypted. Also, the Safe icon appears in the left pane of Windows Explorer, so you can easily access the Safe when managing your files.

The encrypted files can be viewed in different ways by clicking on the buttons on the toolbar or through the menu. You can also sort the files. For example, you can click Size to view files by the amount of disk space they use, or you can click Encrypted to see the files sorted by the date when they were encrypted.

The Safe itself has properties that you can set. These properties affect the way the Safe works. For example, they determine how often the Safe asks you to type in your password. They also specify where encrypted files are stored: in the Safe or in their original location.

In addition to Safe properties, each file in the Safe has its own file properties. When setting properties for the Safe, you must make sure that no files are selected.

You can double-click any encrypted file to decrypt it and then run it using its associated application. For example, if you double-click a file called file.txt, the Safe decrypts the file and then opens it in your default text editor (such as Notepad). This method only works for files in the Safe; it does not work for files that were encrypted in place.
