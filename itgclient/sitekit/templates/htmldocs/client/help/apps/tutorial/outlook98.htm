Securing your E-mail with Microsoft Outlook 2002 and your new Symantec Managed PKI Digital ID

This tutorial will teach you how to use secure e-mail with Outlook 2002 and your Managed PKI Digital ID. Once you have completed these steps, you can digitally sign your e-mail as well as send and receive encrypted messages. You can also bookmark this tutorial for future reference.

If Microsoft Word is set as the e-mail editor:

If you use Microsoft Word as the e-mail editor, it is not possible to add the 'Encrypt' and 'Sign' buttons to the toolbar. You have a choice of signing and encrypting all the e-mail messages, or you can do this on a per e-mail basis.

To sign and encrypt all outgoing e-mail messages: 

  1. In the Tools menu, select Options. The Options dialog box opens.
  2. Select the Security tab.
  3. Select Encrypt contents and attachments for outgoing messages and Add digital signature to outgoing messages.
  4. Click OK to close the Options dialog box.

To sign and encrypt a particular e-mail message: 

  1. Compose the email message.
  2. Click the Options button on the toolbar. The Message Options dialog box opens.
  3. Click Security Settings.
  4. Select Encrypt message contents and attachments and Add digital signature to this message.
  5. Click OK to close the Message Options dialog box.

If native Outlook 2002 e-mail editor is set:

If you use the native Outlook 2002 e-mail editor , Outlook 2002 works exactly like Outlook 98 and Outlook 2000 as described below.

 

Securing your E-mail with Microsoft Outlook 98 and Outlook 2000 and your new Symantec Managed PKI Digital ID

This tutorial will teach you how to use secure e-mail with Outlook 98 and Outlook 2000 and your Managed PKI Digital ID. Once you have completed these steps, you can digitally sign your e-mail as well as send and receive encrypted messages. You can also bookmark this tutorial for future reference.

Simple Steps to Secure E-mail

Configure Outlook to use your new Digital ID.

  1. Choose Tools > Options from the menu.
  2. Select the Security tab.
  3. Click the Change Settings button. See the following figure for help.

  4. In the Change Settings dialog box, click the Choose button next to the Signing Certificate field.

  5. Select the Digital ID that you want to use in Exchange. (Exchange calls Digital IDs certificates.) If you have more than one Digital ID, look at the "Issued to" column and select the ID whose e-mail addresses matches the address you use in Exchange and Outlook. Click OK.
  6. In the Change Settings dialog box, click the other Choose button (next to Encryption Certificate). Choose the Digital ID you want to use when encrypting e-mail messages. This should be the same ID you chose in the previous step.
  7. Click OK to close the dialog boxes.

Your Digital ID is now installed in Outlook. You can set up Outlook to make it easy for you to send signed and encrypted e-mails.

Setting Up Your Toolbar for Secure E-mail

Before you sign and encrypt your e-mail, add signing and encrypting icons to your toolbar:

  1. Open a new mail message by choosing File > New from the menu.
  2. In the message window, go to View > Toolbars > Customize.
  3. Select the "Commands" tab and click on the "Standard" category.
  4. Scroll down the list under Commands and drag and drop the "Encrypt Message Contents" and "Digitally Sign Message" icons into the toolbar. (Note: Drop the icons anywhere to the left of the office assistant icon.)

Digitally Signing Your E-mail

Digitally signing a message lets the recipient know that the message and any attachments really came from you and not an impostor. A digital signature also assures that the message and attachments were not tampered with.

You can manually sign individual messages or you can set up Outlook to automatically sign all of the e-mail messages you send.

Signing Individual Messages

To sign a message, click the Sign button in the toolbar. If you do not see this icon, go to Setting Up Your Toolbar for Secure E-mail.

Automatically Signing All Outgoing Messages

To configure your Outlook to digitally sign all messages:

  1. In Outlook, choose Tools > Options from the menu.
  2. Click the Security tab.
  3. Check the option called "Add digital signature to outgoing messages. Click OK.

Encrypting Your E-mail

E-mail messages are relatively easy to intercept and read. The only way you can be sure that an e-mail message is private is to encrypt it (scramble its contents) so that only the intended recipient can read it. Encrypting messages with Outlook is easy.

To encrypt a message, you need to have a copy of the recipient's Digital ID. Managed PKI automatically publishes your company's internal Digital IDs to the Exchange Directory. Recipients inside your organization should have a Digital ID that can be automatically accessed from the Exchange Directory using the same process you use to address a message to them.

When you receive a signed message from someone outside your organization, you can store the sender's Digital ID in your Personal Address Book or Contacts by right-clicking on the "From" field in the message. You can also search for external Digital IDs from Symantec's Public Certificate Directory by going to: digitalid.verisign.com/services/client/

Encrypting Individual Messages

To manually encrypt a single message:

  1. Click the Encrypt Message button in the message window.
    Note: If you do not see this button, go to Setting Up Your Toolbar for Secure E-mail.
  2. The message and any attachments will be encrypted.
