Open Profile

Please enter the username and password for the PTA profile. This should be the same username and password that you entered when creating the profile at this web-site.

Usernames for all eligible profiles on your computer are displayed in the selection box. You can select the correct username and enter the appropriate password.

Note: If you don't see the username in the selection box, you might have created a Roaming Profile. Please type your username and password to open your profile.
