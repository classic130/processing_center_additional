Managed PKI and Microsoft Exchange Integration Update

The Managed PKI-Exchange Integration Update lets you integrate your Digital IDs into Microsoft Exchange. This helps you manage your IDs as well as other people's IDs.

To install the Integration Desktop:

  1. Click Install Integration Desktop Update. A new page opens. Read the instructions and click OK.
  2. The File Download dialog box appears asking you to download a files called osexsetp.exe. Under the heading, "What would you like to do with this file?," click Save this program to disk.
    Note Running the program is not supported by Symantec. If you select that option, an error message is displayed that the executable is not signed. At this point, you must click Yes and install it. You can start over and re-install at a later time, since following the "Save this Program to Disk" procedure below provides greater security.
  3. Save the program to the Windows Desktop.
  4. In the Windows Desktop directory, double-click osexsetp.exe.
  5. The WinZip Self Extractor dialog box opens. Click Setup to install the software updates.
    Warning: If, after launching the install process, you see a dialog box suggesting that you restart the computer, click No.
  6. After installation is complete, restart your computer before enrolling for your Digital ID, as recommended by the Exchange Configuration dialog box.
