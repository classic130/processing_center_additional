Using Secure E-mail in Outlook

You can use secure e-mail to do the following:

  • Digitally sign a message so that the recipient can verify the message came from you and not an impostor.
  • Encrypt a message so that no one can read it while it travels from your computer to another.
With both features, you can set up Microsoft Outlook to automatically sign or encrypt your messages or you can manually choose to do it on a case-by-case basis.
