Automatically encrypting outgoing messages

You can encrypt all messages that are sent to people whose Digital ID you have stored in Outlook 98.

To automatically encrypt outgoing messages:

  1. Open Outlook Express.
  2. Choose Tools > Options from the main menu (not a message window).
  3. Click on the Security tab.
  4. Check the box labeled Encrypt contents and attachments for outgoing messages.
Any messages you send now will be encrypted, if you have a Digital ID for the person you're writing to.
