Importing and exporting a Digital ID

If you want to use your Digital ID in another application, you can export it from Internet Explorer. Also, if you have another Digital ID that you want to use in Internet Explorer, you can import the ID.

To export your Digital ID to another application:

  1. Start Internet Explorer and choose Tools > Internet Options from the menu.
  2. Click the Content tab and then click the Certificates button.
  3. Select the Digital ID you want to export and click Export.
  4. Enter a password to protect your saved Digital ID. Your Digital ID is saved with a ".p12" file extension. It is recommended that you save your Digital ID on disk and store the disk in a safety deposit box.
To import a Digital ID:
  1. Start Internet Explorer and choose Tools > Internet Options from the menu.
  2. Click the Content tab and then click the Certificates button.
  3. Click Import and then select the file containing your Digital ID. Use the Browse button to select the file. Typically, Digital IDs are exported to files with extensions like .p12 or .PFX.
  4. Enter the password for your private key in the Password field. This is the password you used when you exported the Digital ID.
  5. If you want strong encryption, check the corresponding box and then click Next. You can also mark the private key as exportable.
  6. Select the certificate store where you want your Digital ID to be saved. If you are not sure which option to use, select the Automatic certificate store. The term certificate is simply another name for Digital ID. Click Next.
  7. You get a message confirming the import. Click the Finish button.
