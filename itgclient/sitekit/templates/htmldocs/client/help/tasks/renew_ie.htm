Renewing a Digital ID

You can renew valid or expired Digital IDs so that they are valid for another year. Revoked IDs cannot be renewed.

Typically, the Digital ID center sends you an e-mail message letting you know that your Digital ID is about to expire. The e-mail message contains the URL to the renewal page in the Digital ID Center.

To renew a Digital ID:

  1. Go to the Digital ID Center and click the Renew button.
  2. On the Web page that appears, check the options that you want. If you want to protect your private key, check that box. It is highly recommended that you check this option. You will be prompted for a password that you must enter when you want to use your new Digital ID.
  3. Check the box to publish the public part of your Digital ID to a directory. This is entirely optional, but having your Digital ID in a directory is perfectly safe. Other people can look up your ID in the directory and send you encrypted messages.
  4. Click Renew.
  5. You will see a dialog box asking you to select the ID you want to renew. Select it and click Renew. Your Digital ID is automatically created and installed for you.
