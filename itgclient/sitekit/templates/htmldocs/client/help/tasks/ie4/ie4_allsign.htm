Automatically signing all outgoing messages using Outlook 98

To configure Outlook to automatically sign outgoing messages:

  1. In Outlook Express, choose Tools > Options from the main window menu (not a message window).
  2. Click on the Security tab.
  3. Check the box labled "Add digital signature to outgoing messages".

Any messages you send now will be digitally signed.
