What is a PTA Profile?

A Personal Trust Agent Profile is a credential store that stores your Digital IDs, Digital ID preferences, application preferences, etc. When you enroll for the Digital ID the first time you may be asked to create a PTA profile by asking you to enter a username and password. You will need to remember this username and password to use the Digital IDs in the profile from now on. Note that this password is used to secure your profile and this password is not known to the web-site.

Generally, the profile will be created on your local machine and you need to access the web-site from that machine only. Websites can optionally create Roaming profiles that can be securely accessed from any computer.

You can renew the Digital IDs in the profile by clicking the Renew link on the Digital ID Center page. You can change the password securing your profile by clicking on the "Change Profile Password" link on the Digital ID Center page.
