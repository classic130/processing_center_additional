Importing and Exporting a Digital ID

If you want to use your Digital ID in another application, you can export it from Netscape Communicator. Also, if you have another Digital ID that you want to use in Communicator, you can import the ID.

To import your Digital ID to another application:

  1. In Communicator, click the Security button on the main button bar, or choose Communicator > Security Info from the menu.
  2. Under Certificates, click the Yours link.
  3. Click on the Import a Certificate button near the bottom of the screen. You might need to scroll down to see this option.
  4. Select the Digital ID file to import and click OK.
  5. Enter the private key password. This is the same password you used when you exported the Digital ID.
Communicator automatically adds your Digital ID to its certificate database. (Reminder: some applications call Digital IDs certificates.)

To export a Digital ID:

  1. Follow steps 1 and 2 from the importing section, and then click Export a Certificate.
  2. Enter your Netscape certificate password (usually this is the same as your private key) and then specify a password to protect your Digital ID export file. You will be asked to confirm the export file password. Remember this password because you need to enter it when importing your Digital ID into other applications.
  3. Your Digital ID is saved with a ".p12" file extension. It is recommended that you save your Digital ID on disk and store the disk in a safety deposit box.
