Personal Trust Agent overview

Some web sites use a Personal Trust Agent to help use your Digital ID. The Personal Trust Agent lets you easily perform secure transactions over the Web. Because the transactions are scrambled (encrypted), no one can access your private information.

Also, your Digital ID helps protect you, so that only you can get to your private information. Your Digital ID proves your identity to the Web site, meaning no one else can pose as you and get your information from the Web site. And, the Personal Trust Agent makes it easy for you to use, because it sends you a Digital ID and then remembers which Digital ID it needs. This eliminates the hassle of having to remember which Digital ID is used for which Web site.
