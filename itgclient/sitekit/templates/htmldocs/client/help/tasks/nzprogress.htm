Symantec Digital Notarization Client 

Symantec’ s Digital Notarization client allows users to digitally notarize documents/content. The result of the notarization is a digital receipt that contains non-repudiable proof that the document/contents was digitally notarized at a given time. The client can be used via the right click context menu interface or through a web page.

Using the client from the Windows desktop:

  1. To perform digital notarization right click on the document, this brings up the (Symantec Digital Notarization) context menu. Click on this option.
  2. The digital notarization client performs the digital notarization operation and saves a receipt in the same location as the document. The receipt has the ‘.rcpt’ extension.
  3. Double click on the receipt file to verify it. If you notarize the same document again a new receipt file will be created.
