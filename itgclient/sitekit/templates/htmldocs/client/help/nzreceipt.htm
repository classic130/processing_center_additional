Symantec Digital Notarization Receipt

The receipt contains information that provides undeniable proof that a document or some content existed when the digital notarization was requested.

Each receipt contains the following information:

  1. A fingerprint of the document that was digitally notarized.
  2. Information about the company or the individual that requested digital notarization.
  3. A timestamp giving information of the date and time the document was digitally notarized.

The Digital Notarization Client shows the following information for each receipt:

  1. For: The name of the user that requested digital notarization.
  2. Title:The filename or a description of the document that was digitally notarized.
  3. Date: The date the document was digitally notarized.
  4. Time:The time the document was digitally notarized.
  5. Valid/Invalid: Indicates whether the receipt is still valid. Making any changes to a digitally notarized document invalidates the receipt.
  6. Web Service : Click this link to verify your document with the receipt stored in Symantec’s database. (This link only appears if the receipt is valid.)
