Downloading a Digital ID

This page lets you download a Digital ID.

To download the Digital ID:

  1. Select the Digital ID format you want. The formats are as follows:
    • Someone Else's ID for Netscape is the format you want if you are downloading someone's Digital ID and you use a Netscape browser. Choose this if you want to send someone secure (encrypted) e-mail on Netscape.
    • Someone Else's ID for Microsoft IE/Outlook Express/Outlook is the format you want if you are downloading someone's Digital ID and you use a Microsoft browser or Outlook to send someone secure (encrypted) e-mail with a Microsoft mail tool.
    • My ID for Netscape lets you download your Digital ID if you use Netscape's browser.
    • My ID for Microsoft IE/Outlook Express/Outlook lets you download your Digital ID if you use Microsoft's products.
    • S/MIME Format (Binary PKCS#7) is a generic format for use in applications other than Netscape's or Microsoft's products. Select this option only if you are certain you need this format.
  2. Click the Submit button.

If you are downloading your own Digital ID, you see a dialog box asking for your private key password. Type in the password and click OK. The Digital ID is saved to your browser. If your browser does not have a private key that matches the Digital ID you are downloading, a message opens to indicate the ID could not be downloaded.

If you are downloading someone else's Digital ID, you see a dialog box asking you to confirm that you want to download the ID (also called a certificate in some Web browsers). Click OK.
