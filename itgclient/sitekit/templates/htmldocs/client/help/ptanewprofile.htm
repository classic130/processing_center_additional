Creating a New Profile

When you enroll for a Digital ID you will be asked to create a new PTA profile. You will need to enter a username and password to create the profile. The password specifications are mentioned on the Create Profile Dialog box. Note that this password is used to secure the profile on your machine and will not be sent to any web-site. You will need to remember this username and password to use the Digital IDs in your profile.

A request for a Digital ID will be generated and sent to the web-site. If your request is approved a Digital ID is automatically installed in your profile. Otherwise, you will receive an email with instructions on how to install the Digital ID in your profile. In this case, you might be asked to enter the username and password that you entered when you created the profile.
