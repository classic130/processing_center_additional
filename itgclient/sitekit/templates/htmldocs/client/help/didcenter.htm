Using Digital ID Center

The Digital ID Center provides the following services:

Enrolling for a Digital ID lets you request a Digital ID. You will get confirmation via e-mail.

Picking up a Digital ID lets you receive and install your Digital ID after you receive your confirmation e-mail.

Searching for a Digital ID lets you find Digital IDs that belong to coworkers or associates so you can send them encrypted e-mail.

Revoking a Digital ID lets you cancel an ID that you no longer use. This includes any Digital IDs that might have been lost or compromised.

Renewing a Digital ID lets you receive a new Digital ID that contains the same information as the last one except it has a new expiration date.
