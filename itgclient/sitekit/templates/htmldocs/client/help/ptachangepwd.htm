Change your Profile password

You can change the password for your PTA profile using the 'Change Password' feature.

To change the password:

  1. You can go to the Change Password page by clicking here.
  2. (Optional) If your profile is not open, you will have to enter the username and password to open your profile.
  3. Now you will be asked to enter your existing password, and your new password. You can now click the "Change" button to change the password.
  4. From now on, you will need to use the profile username and new password to use the Digital IDs.
